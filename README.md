# ⚡ Convert

Conversor de moedas simples e elegante, com interface dark e cotações em tempo real — converte qualquer valor para **Reais (BRL)**.

---

## 📸 Preview

![Preview do site Convert](./convert-template-main/img/preview.png)

---

## ✨ Funcionalidades

- Conversão de moedas estrangeiras para BRL (Reais)
- Suporte a múltiplas moedas via seletor
- Parâmetros de URL para pré-preencher valor e moeda (`?amount=9&currency=USD`)
- Interface dark minimalista com fundo animado estilo gráfico financeiro

---

## 🚀 Como usar

### Localmente

1. Clone o repositório:
   ```bash
   git clone https://github.com/seu-usuario/convert-template.git
   cd convert-template/convert-template-main
   ```

2. Abra o `index.html` com um servidor local (ex: Live Server no VS Code):
   ```
   http://127.0.0.1:5500/convert-template/convert-template-main/index.html
   ```

### Via URL com parâmetros

Você pode pré-preencher o formulário diretamente pela URL:

```
index.html?amount=9&currency=USD
```

| Parâmetro  | Descrição                        | Exemplo   |
|------------|----------------------------------|-----------|
| `amount`   | Valor a ser convertido           | `9`       |
| `currency` | Código da moeda de origem (ISO)  | `USD`     |
